Download the ZIP file.
Copy all these programs and the launcher file in the folder where the images to be analysed are saved.
Edit the launcher file as per your test conditions keeping the format and arrangement to be same.
In the launcher file, the first line (no need to edit) is the heading showing the name of parameter.
The second line is the value corresponding to the parameter written in the same order over it.
This line needs to be edited based on the images in the current folder.
Open the folder in MATLAB and run these programs one by one, following the steps mentioned in the paper.
